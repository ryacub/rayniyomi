# Rayniyomi

Fork of Aniyomi with anime/manga tracking, reading, and watching.

[![CI](https://img.shields.io/github/actions/workflow/status/ryacub/rayniyomi/build_push.yml?labelColor=27303D)](https://github.com/ryacub/rayniyomi/actions/workflows/build_push.yml)

</div>

## Why Rayniyomi?

- **Auto-scroll webtoons** — hands-free reading with tap-to-pause
- **PIN lock** — secure your stuff with actual security, not just a pattern
- **Resumable downloads** — downloads that actually resume when interrupted
- **Light novel support** — yes, LN readers welcome
- **AI translation toggle** — read untranslated stuff on the fly
- **8 trackers** — MyAnimeList, AniList, Kitsu, Bangumi, etc.
- **Categories with search** — organize that 500-entry library

## Features

- Local reading/watching
- Configurable reader (multiple viewers, directions)
- Configurable player (mpv-based)
- Schedule updates
- Cloud backups
- Dark/light themes
